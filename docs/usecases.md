# Use cases

## 1. Pure-ABM baseline (no LLM)

The deterministic calibration path of the paper. No LLM is contacted.

```bash
cargo run --release -- run --dataset metoo --abm bc --core-ratio 0.0 \
    --n-agents 1000 --steps 14 --network ba --seed 42
```

Compare the four opinion-dynamics models — BC/HK converge toward consensus (low `polarization`), SJ/Lorenz drive two-pole splits (high `polarization`, large `mobilized`):

```bash
for m in bc hk sj lorenz; do
  cargo run --release -- run --abm $m --core-ratio 0.0 --n-agents 1000 --steps 14 --seed 42
done
```

## 2. Hybrid (LLM core + ABM periphery)

Needs a local Ollama with a pulled model. The core tier's LLM-driven attitudes propagate to the ordinary tier (one-way coupling), correcting the pure-ABM trend.

```bash
ollama pull llama3.2:latest
OLLAMA_HOST=http://localhost:11434 OLLAMA_MODEL=llama3.2:latest \
cargo run --release -- run --dataset metoo --abm bc --core-ratio 0.3 \
    --n-agents 1000 --steps 14 --network ba --llm-temperature 0 --seed 42
```

The first run populates `.llm_cache/cache.json`; rerunning the same command replays from cache (100% hit, no LLM calls) and reproduces the result exactly.

## 3. Offline hybrid smoke (no live LLM)

For CI / network-isolated environments: a `ScriptedClient` mock drives the core tier through the full hybrid pipeline.

```bash
cargo run --release --example mock_smoke -- results
```

## 4. Sensitivity sweep

```bash
cargo run --release -- sweep \
    --core-ratio-min 0.0 --core-ratio-max 0.5 --core-ratio-step 0.1 \
    --abm-values bc,hk,sj,lorenz --network-values ba,ws,er --runs 10 --seed 42
uv run hisim-tools visualize-sweep
```

Expected qualitative findings: a small core-ratio already corrects the macro trend; BA amplifies core influence more than WS/ER; SJ/Lorenz polarize while BC/HK reach consensus.

## 5. Reproduce the paper (Table 2/3 + SoMoSiMu-Bench)

One command reproduces the headline results: the hybrid-vs-pure-ABM contrast across all four ordinary-tier models (Table 3) and the SoMoSiMu-Bench movement-metric alignment per dataset (Table 2).

```bash
# Fully offline: the pure-ABM arm + bench alignment make no LLM calls;
# --mock drives the hybrid arm with a scripted client.
cargo run --release -- reproduce --mock --seed 42

# Render the report + figures (also drives the binary with --run):
uv run hisim-tools reproduce --run --mock
```

`reproduce_summary.json` records observed-vs-paper anchors with PASS/off bands (pure-ABM = zero LLM calls; Lorenz/SJ polarize above BC; BC/HK stay in consensus; the LLM core amplifies mobilization) and the per-movement bench alignment. The bench reference is a **calibrated synthetic** curve — the raw SoMoSiMu-Bench dataset is not bundled, so the comparison checks qualitative agreement, not exact numbers.

## 6. External-LLM stance annotation

By default a core post's stance is mapped to an attitude by a deterministic keyword classifier (bit-identical, no extra LLM calls). Opt into the paper's external-LLM annotation with `--stance-annotator llm` (needs a live backend): the core LLM classifies each post's stance on a 5-point scale, mapped to `[-1, 1]` and cached.

```bash
OLLAMA_MODEL=llama3.2:latest cargo run --release -- run \
    --dataset metoo --abm bc --core-ratio 0.3 \
    --stance-annotator llm --llm-temperature 0 --seed 42
```
